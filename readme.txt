Лабораторная работа №2
Тема: Линейный вычислительный процесс

Цели и задачи работы: изучение функций ввода-вывода данных, программирование вычисления значения выражения.
В проекте реализованы две задачи с использованием JavaScript: вычисление квадрата периметра квадрата и нахождение количества полных метров в расстоянии в сантиметрах.

Содержит файлы:
- index.html (основной файл)
- styles.css (стили)
- script.js (логика)
- README.txt (описание)
